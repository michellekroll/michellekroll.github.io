Berkeley Reentry Student Program Web Application

Worked with a team of 5 peers following Agile software development methodology to develop a Ruby on Rails web application for the UC Berkeley Reentry Student Program, impacting over 1,000 undergraduate students. Created functionality for checking into study spaces using Google OAuth, creating appointments with advisors on Google Calendar, and dynamically updating student profiles. Conducted weekly customer meetings to drive feature planning and user stories, underwent multiple development iterations, and utilized Pivotal Tracker and GitHub for team collaboration.

Trips

Utilized Firebase and Java to create a dynamic, interactive vacation itinerary that responds to real-time user-generated data across different accounts on the app, and collaboratively ideated with team of 4 engineers and validated user needs while maintaining project documentation throughout a four-week process.

BearMaps

Implemented a web browser mapping application similar to Google Maps using the A* search algorithm and utilized data structures such as KDTrees, Heaps, and Priority Queues.

Build Your Own World

Worked in collaboration with one other student to create a 2D tile-based random world exploration engine, which involved implementing UI with Heads Up Display, keyboard mode, and string mode where users can interact with a moving avatar. Utilized KDTrees and A* search algorithm to create paths for avatar to navigate randomly generated rooms and hallways in the world.

MDB Socials

Implemented a mobile app in Swift using Xcode to promote socialization in MDB that supports over 80 members of the club in creating and sharing social events. Utilized Firebase Authentication and Cloud Firestore to create email sign-in and register pages, as well as a social events feed.

Scheme

Created interpreter for the Scheme programming language using Python and included capabilities such as reading Scheme expressions, symbol evaluation, calling built-in procedures, and evaluation of special forms.
